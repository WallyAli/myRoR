# How To Install

## Install RVM

### Check Version
  ruby -v 

### Install RVM:
  \curl -L https://get.rvm.io | bash
  
### Make update-to-date 
   rvm get stable

### Install Ruby
   rvm install 2.0.0 
   
* check this stackoverflow post if you encounter some errors:
